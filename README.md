\# QuickPOS Landing Page



!\[QuickPOS CI/CD Pipeline](https://github.com/UsMaNBAjWaa/quickpos-landing-page/actions/workflows/ci.yml/badge.svg)



\## Project Overview



QuickPOS is a responsive Point of Sale (POS) landing page developed using PHP.



This project includes:



\- Responsive landing page

\- Contact form validation

\- PHPUnit automated testing

\- PHP syntax validation

\- PHP code quality checks

\- GitHub Actions CI/CD pipeline

\- Branch protection workflow



\---



\## Features



\- Hero Section

\- Features Section

\- Pricing Plans

\- Contact Form

\- Footer

\- Responsive Design



\---



\## Automated Testing



Implemented automated test cases for:



\- Empty form validation

\- Invalid email validation

\- Successful form submission

\- Landing page availability



\---



\## CI/CD Pipeline



GitHub Actions pipeline includes:



\- Code Quality Check

\- Automated Tests

\- PHP Syntax Validation

\- Artifact Upload

\- Parallel CI Jobs

\- Jira Commit Validation



\---



\## Branch Protection Rules



The `develop` branch is protected.



Rules enabled:



\- Pull request required before merge

\- CI checks must pass before merge

\- Direct commits restricted

\- Jira ticket IDs required in commit messages
